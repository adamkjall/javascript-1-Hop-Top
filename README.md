# javascript-1-Hop-Top

Slutprojekt i kursen Javascript 1, Medieinstituet.

Projektet -  att skapa ett spel, i grupp, med syfte att väva samman allt från kursen Javascript 1. 
Spelet är uppsatt med en kodbas som använder sig av TypeScript och P5. 
Projektet är indelat i tre milstolpar: idébeskrivning, mellandagsanvstämning samt presentation och inlämning.

Spelidé - spelet utgår från det gamla spelet "Hop to the top", och spelet går ut på att en spelare studsar på block och ska uppåt så långt möjligt. Spelplanen består av olika levels, och spelaren dör om denne ramlar nedåt.
